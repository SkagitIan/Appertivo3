Embed Widget

Add the Appertivo widget to your restaurant website to showcase daily specials and capture customer emails.

Embed Code

Copy this code and paste it into your website where you want the widget to appear:

<div id="appertivo-widget"></div> <script src="{{ widget_js_url }}"></script>

Features:

  • • Click tracking for analytics

How to Embed on Different Platforms

Follow the steps for your website builder or CMS:

Wix

  1. In your Wix dashboard, go to the page where you want the widget.
  2. Click Add → Embed → Embed a Widget.
  3. Choose Embed HTML and paste the Appertivo code snippet.
  4. Resize the embed box as needed and publish your site.

Shopify

  1. From your Shopify admin, go to Online Store → Themes.
  2. Click Customize on your active theme.
  3. Navigate to the section (Homepage, Product page, etc.).
  4. Add a Custom HTML block and paste the Appertivo embed code.
  5. Save changes and preview your store.

Square (Weebly)

  1. Open your Square Online editor.
  2. Drag an Embed Code element onto the page where you want the widget.
  3. Click inside the element, then paste the Appertivo code snippet.
  4. Adjust sizing and publish your changes.

Generic Website (HTML / WordPress, etc.)

  1. Open your site’s HTML editor (or use a code block if in WordPress or similar).
  2. Paste the Appertivo code snippet where you want the widget to appear:
    3. <div id="appertivo-widget"></div>
<script src="{{ widget_js_url }}"></script>
  3. Save and publish your page.
